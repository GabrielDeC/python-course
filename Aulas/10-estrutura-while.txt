A estrutura While é mais flexível que a estrutura For

Sua estrutura é uma For "sem limite" e só para sob uma condição

Chama-se estrutura de repetição com variável lógica

while not x:
    if c
        passo
    if sc
        pula
    if moeda
        pega
pega


A estrutura de repetição While sempre será executado até finalizar a condição definida, no entanto, se for necessário parar no meio da condição definida pode ser utilizada a quebra da condição.

Exemplo:

while True:
    if c:
        x
    if a:
        y
    if b:
        z
    break
pega


n = cont = 1

while True:
    n = int(input('Digite um número: '))
    if n == 999:
        break
    cont += n
print(cont)
